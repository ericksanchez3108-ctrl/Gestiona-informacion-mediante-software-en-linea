# Gestiona-informacion-mediante-software-en-linea
investigacion 1
GitHub es una plataforma de desarrollo colaborativo que utiliza el sistema de control de versiones Git. Se ha consolidado como el estándar de la industria para alojar proyectos de código abierto y privados, permitiendo a los desarrolladores gestionar cambios en el software de forma organizada.

Características Principales
Repositorios: Espacios donde se almacena el código, el historial de cambios y la documentación.

Pull Requests: Herramienta que permite a los desarrolladores proponer cambios en un proyecto, facilitando la revisión de código antes de integrarlo.

GitHub Actions: Un sistema de automatización para CI/CD (Integración y Despliegue Continuo) que ejecuta pruebas y despliegues automáticamente.

Issues y Proyectos: Funcionalidades de gestión de tareas para rastrear errores, sugerencias y organizar el flujo de trabajo mediante tableros tipo Kanban.

Ventajas
Colaboración Global: Facilita que personas de cualquier parte del mundo contribuyan a un mismo proyecto simultáneamente sin sobrescribir el trabajo ajeno.

Portafolio Profesional: Funciona como un currículum social donde los reclutadores pueden ver la calidad y frecuencia del código de un programador.

Comunidad y Documentación: Alberga la mayor colección de código abierto del mundo, lo que permite aprender de otros y reutilizar librerías existentes.

Seguridad: Ofrece herramientas automáticas para detectar vulnerabilidades en las dependencias del código.

Desventajas
Curva de Aprendizaje: Para principiantes, entender la lógica de Git (commits, push, pull, merges) puede resultar confuso al inicio.

Privacidad en Planes Gratuitos: Aunque ya permite repositorios privados gratuitos, históricamente estaba muy enfocado en lo público, y ciertas funciones avanzadas requieren suscripciones de pago.

Dependencia de la Nube: Al ser una plataforma centralizada, si los servidores de GitHub caen, el flujo de trabajo colaborativo se detiene temporalmente, aunque Git siga funcionando de forma local.

Límites de Almacenamiento: No es ideal para gestionar archivos binarios extremadamente pesados (como videos o assets de juegos) sin recurrir a extensiones adicionales como Git LFS.
